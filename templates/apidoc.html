Todos WebApp API

You can use the Todos WebApp API to get details of your todos in a json format

URL

http://sggts04.pythonanywhere.com/api?user=your_username&pass=your_password

where your_username is the username associated with your account and your_password is the password associated with your account

Output Format

{
  "Todo Title": [
    "Done or Not Done", 
    "Todo Description", 
    "Date of Adding Todo", 
    "Date by which Todo should be completed", 
    "Date on which Todo was completed"
  ]
}

Example Output

{
  "Algebra": [
    "Done", 
    "Study and Complete the Assignment",

  ],

}
