# Todos WebApp
This is a Todos WebApp built in Flask, currently with no user system, and no database, only json files.

## Installation
```
git clone https://github.com/sggts04/flask_todos
cd flask_todos
pip install flask
python app.py
```
The site will be available on http://127.0.0.1:5000/ then.
